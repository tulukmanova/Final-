# Final Project — Bootstrap Base

This repository contains the Bootstrap 5 base for my final project.

## How to use
- Replace placeholder content in `index.html`, `pages/*`, and images in `assets/img/`.
- Add your Module 1–5 content into the corresponding sections.

## Run locally
Just open `index.html` in a browser.

## Deploy with GitHub Pages
Enable Pages in **Settings → Pages**, choose the `main` branch and `/ (root)`.
